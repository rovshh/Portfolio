World Cup Football Tournament System

University project for Python Programming course. In this project my team and I developed a programm with GUI, 
in which user selects football teams for the roster of World Cup. Then, the system calculates winners for each 
match and forwards them to the next stage of tournament. It is done from group's stage to semifinal and final 
matches.
Main objective of this project was to improve and test our programming skills with emphasizes on OOP and GUI in 
Python.
